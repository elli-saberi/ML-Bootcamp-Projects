# ML Bootcamp Projects

This repository showcases a series of Machine Learning mini-projects completed during the bootcamp journey.  
Each project focuses on essential data science tasks including data analysis, feature engineering, statistical testing, and predictive modeling using Python and Scikit-learn.


---

## Project List

● **project-01-covid19-pandas-analysis** – Exploratory data analysis of COVID-19 datasets using Pandas. 

● **project-02-city-distance-calculator** – Python-based distance computation between city coordinates. 

● **project-03-olympic-visualization** – Visualization of Olympic statistics with Matplotlib and Seaborn.

● **project-04-boston-housing-statistics** – Statistical testing in the Boston Housing dataset (Shapiro & Mann-Whitney tests). 

● **project-05-student-engagement-analysis** – Data loading and preliminary analysis of engagement ratios.

● **project-06-feature-engineering-energy** – Feature engineering and polynomial transformations on energy consumption data. 

● **project-07-modeling-life-expectancy** – Regression modeling of global life expectancy data with Polynomial Features.  

● **project-08-blood-donation-prediction** – Classification using Logistic Regression and SMOTE balancing.  

● **project-09-bank-marketing-classification** – Predicting customer subscription using Random Forest and F1-score evaluation.  

● **project-10-iris-classification** – Ensemble learning applied to Iris dataset with Decision Trees and Random Forest.

---

## Tech Stack

● **Languages:** Python  
● **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn  
● **Concepts:** Regression, Classification, Statistical Testing, Feature Engineering
